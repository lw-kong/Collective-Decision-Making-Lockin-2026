# Collective-Decision-Making-Lockin-2026
[![DOI](https://zenodo.org/badge/1216560461.svg)](https://doi.org/10.5281/zenodo.19674894)

## Directories
### functions
This folder contains functions used in many other scripts, including those for generating communication networks and the core agent-based simulation of our collective decision-making framework. Make sure you always add this folder to the path.

### analyses
- Core: Core simulations of the DeGroot model, Wisdom of the Crowd model, and quantized message models.
- Limited_Attention: Simulations of the limited attention (top-L) model.
- Mean_Field: Simulations of the mean-field results of the quantized messages models and the limited attention model.
- Bio_Ants, Bio_RingAttractor, Bio_Neurons: Simulations of the three biologically grounded models (Fig. 4 in the main text).
- Feedback_Payoff: Simulations with the payoff-dependent feedback mechanism (Fig. 5 in the main text).
